Image Carousel - Angular Integration Notes

Build Techniques

This carousel displays a scrolling row of images fetched from a PHP API. It works across desktop and mobile, supports touch gestures, and includes a built-in lightbox viewer.

How it Works
Data Source
  • Images are stored on the server and listed by a PHP script (carousel_api.php).
  • The script takes a category (e.g. ?category=ai) and returns an array of image URLs (JSON).
API Calls from Angular
  • Angular ApiService requests the PHP script with HttpClient.
  • proxy.conf.json is used in local dev so calls like /api/carousel_api.php avoid CORS.
